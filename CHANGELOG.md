# Changelog

## [1.0.0] - 2024

### Добавлено
- ✅ Telegram бот для анализа голоса через голосовые сообщения
- ✅ Веб-интерфейс для просмотра результатов всех пользователей
- ✅ DSI (Dysphonia Severity Index) - расчет и интерпретация
- ✅ API сервер с экспортом результатов (CSV, JSON, HTML)
- ✅ Полный отчет в Telegram после обработки голосового сообщения
- ✅ Docker контейнеризация для простого развертывания
- ✅ Автоматическое сохранение результатов в `results.json`
- ✅ Статистика: общее количество анализов, пользователей, анализов за неделю

### Функциональность
- Извлечение акустических признаков (Jitter, Shimmer, HNR, F0, MPT и др.)
- Анализ симптомов болезни Паркинсона (оценка 0-3)
- Оценка риска ПД на основе пороговых значений
- Детальный текстовый отчет с рекомендациями

### Технические детали
- Python 3.11+
- Flask API сервер
- Aiogram для Telegram бота
- Parselmouth для анализа голоса
- Librosa для обработки аудио
- Docker и docker-compose для контейнеризации
