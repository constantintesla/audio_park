# Changelog

## [1.1.0] - 2025-01-19

### Изменено
- ✅ **Убраны капы и фолбеки из параметров** - теперь возвращаются настоящие значения без искусственных ограничений:
  - HNR: убраны капы на 8.0 дБ (минимум) и 25.0 дБ (максимум)
  - Jitter: убраны капы на 5.0% (максимум) и 0.01% (минимум)
  - Shimmer: убран кап на 50.0% (максимум)
  - Все параметры теперь отражают реальные результаты расчета

### Добавлено
- ✅ **Функция пересчета всех результатов** - кнопка в веб-интерфейсе для пересчета всех сохраненных результатов с обновленными алгоритмами
- ✅ **Реорганизован веб-интерфейс**:
  - Главная страница (`/`): сводная таблица результатов + отображение одного результата при клике
  - Страница списка результатов (`/results`): полный список всех результатов в виде карточек
  - Страница визуализации (`/visualization?index=X`): автоматический выбор записи по индексу из URL
- ✅ Автоматическая прокрутка и фокус на отчет при клике на строку таблицы
- ✅ Кнопка визуализации в карточках результатов для прямого перехода

### Улучшено
- Более точные результаты анализа за счет использования реальных значений параметров
- Улучшенная навигация между страницами
- Оптимизированная структура интерфейса

## [1.0.0] - 2024

### Добавлено
- ✅ Telegram бот для анализа голоса через голосовые сообщения
- ✅ Веб-интерфейс для просмотра результатов всех пользователей
- ✅ DSI (Dysphonia Severity Index) - расчет и интерпретация
- ✅ API сервер с экспортом результатов (CSV, JSON, HTML)
- ✅ Полный отчет в Telegram после обработки голосового сообщения
- ✅ Docker контейнеризация для простого развертывания
- ✅ Автоматическое сохранение результатов в `results.json`
- ✅ Статистика: общее количество анализов, пользователей, анализов за неделю
- ✅ Страница визуализации параметров с графиками и аудио

### Функциональность
- Извлечение акустических признаков (Jitter, Shimmer, HNR, F0, MPT и др.)
- Анализ симптомов болезни Паркинсона (оценка 0-3)
- Оценка риска ПД на основе пороговых значений
- Детальный текстовый отчет с рекомендациями
- Сохранение сырых данных (аудиофайлы, waveform, спектрограммы)

### Технические детали
- Python 3.11+
- Flask API сервер
- Aiogram для Telegram бота
- Parselmouth для анализа голоса
- Librosa для обработки аудио
- Docker и docker-compose для контейнеризации
